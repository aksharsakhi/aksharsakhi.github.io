## 🌐 Deploy to GitHub Pages

### Step 1: Create GitHub Repository

```bash
# Navigate to project folder
cd /Users/akshar/Desktop/portfolio-sheela

# Initialize git
git init

# Add all files
git add .

# Commit
git commit -m "Initial commit: Portfolio website"
```

### Step 2: Create GitHub Repo

1. Go to https://github.com/new
2. Repository name: `portfolio` (or `sheela-portfolio`)
3. Make it **Public**
4. Click **Create repository**

### Step 3: Push to GitHub

```bash
# Add remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Step 4: Enable GitHub Pages

1. Go to your repo on GitHub
2. Click **Settings** → **Pages** (left sidebar)
3. Under "Source", select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Click **Save**

### Step 5: Access Your Site

After 2-3 minutes, your site will be live at:
```
https://YOUR_USERNAME.github.io/portfolio/
```

## 🎨 Customization

### Change Colors

Edit CSS variables in `css/styles.css`:

```css
:root {
    --accent-orange: #FF9500;      /* Main orange */
    --primary-dark: #1a1a1a;       /* Dark text */
    --bg-light: #F5F7FA;           /* Light background */
}
```

### Update Content

All content is in `index.html`. Just find and replace:
- Text content
- Links (GitHub, LinkedIn, etc.)
- Project details
- Timeline items

### Adjust 3D Particles

In `js/main.js`:
- Change particle count (currently 3000)
- Modify particle colors
- Adjust animation speed
